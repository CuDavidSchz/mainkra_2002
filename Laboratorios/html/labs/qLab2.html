Preguntas Laboratorio 2 - CSS

  1. Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

    Es una regla de estilo que no debería ser usada muy seguido porque quiere decir que la jerarquía de elementos está mal distribuida y puede hacer más difícil el mantemiento.

  2. Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

    Puede causar que el sitio tarde demasiado en abrir y procesarse. Además, la mayoría de efectos pueden ser logrados através de CSS puro o con ayuda de frameworks de estilo.

    Sin embargo, aveces es necesaria para darle ciertos efectos a la página. En estos casos se recomienda hacer la imagen a la medida para facilitar la implementación.

  3. Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

    Cada una tiene sus usos específicos:

    %: Sirve para que las proporciones de los elementos sean responsivas. Esto quiere decir que se adaptan a varios tamaños de pantallas.

    px: Es para los elementos que necesitan medidas fijas. Algunos usos serían los logos.

    pt: No se recomienda usar en el diseño web, sino en impresión.

  4. ¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

    Ayuda porque tiene menos caracteres innecesarios lo que facilita la transferencia de la hoja de estilos desde donde está alojado hasta para la lectura del mismo para la interpretación.
