Preguntas Laboratorio 4 - JavaScript

  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    Diferencias:

    • Java es un lenguaje compilado y fuertemente tipado, lo que quiere decir que se requiere especificar el tipo de dato que se está manejanfo al momento de declarar una variable. Por otro lado, JavaScript es un leguaje interpretado y dinámicamente tipado.
    • Java es un lenguaje orientado a objetos puro, mientras que JavaScript es un lenguaje que emula la programación orientada a objetos con prototipos.
    • Java es usado para crear aplicaciones de escritorio, móviles y backend. Por otro lado JavaScript se enfoca en el desarrollo Web.

    Semejanzas:

    • Compatibilidad: ambos lenguajes pueden ejecutarse en el navegador.
    • Nombre: ambos lenguajes tienen "Java" en el nombre, pero hasta ahí jaja.
  2. ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

    Algunos métodos disponibles para Date son:

    • getYear(): Devuelve el año de la fecha.
    • getMonth(): Devuelve el mes del año. Desde enero como 0 hasta diciembre como 11.
    • getDay(): Devuelve el día de la semana. Desde domingo como 0 y sábado como 6.
    • getDate(): Devuelve el día del mes.
    • setHours(): Cambia la hora, minutos, segundos y milisegundos de la fecha a la hora local.
  3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

    Algunos métodos de los arreglos son:

    • pop(): Elimina el último elemento y lo devuelve.
    • shift(): Elimina el primer elemento y lo devuelve.
    • unshift(): Añadir un elemento al principio del arreglo.
    • push(elemento): Agrega un elemento al final del arreglo.
    • indexOf(elemento): Encontrar el índice de un elemento en el arreglo.
  4. ¿Cómo se declara una variable con alcance local dentro de una función?
    • Se usa let para que el alcance sólo sea dentro de la función.
  5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    Pueden ocurrir algunos errores como:

    • Consumo de memoria: la variable seguirá en la memoria de la computadora porque se supone que será necesitada en el alcance global del programa.
    • Detección de errores: en caso de ser usada en varias funciones que se ejecutan al mismo tiempo puede ocasionar errores difíciles de detectar debido a la sobreescritura de la misma.
